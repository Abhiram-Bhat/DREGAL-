Simple E-Commerce Website
Description
This is a basic e-commerce website project built with HTML, CSS, and JavaScript. It features a product display page,
Features
View product grid with images
Filter products by category (Tech, Fashion, Lifestyle)
Add items to the shopping cart
Increase or decrease item quantities in the cart
Automatic total price calculation
Responsive design for mobile and desktop
Technologies Used
HTML5
CSS3
Vanilla JavaScript
Project Structure
The project consists of three files located in the main directory:

index.html: Contains the structure of the webpage.
style.css: Contains all the styling and design logic.
script.js: Contains the logic for the cart and product display.
